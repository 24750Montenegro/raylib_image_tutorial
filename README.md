# raylib_image_tutorial

Relleno de polígonos con raylib (Rust): relleno por scanline (regla par-impar)
y borde con el algoritmo de Bresenham.

## Polígonos

- Polígono 1: amarillo con orilla blanca
- Polígono 2: azul con orilla blanca
- Polígono 3: rojo con orilla blanca
- Polígono 4: verde con orilla blanca
- polígono 5: agujero que no se pinta


## Uso

```
cargo run
```

Genera `out.png` y `out.bmp`.
